# Propello AI — Real Estate Chatbot

An AI-powered real estate sales chatbot built with Python FastAPI and Groq (OpenAI-compatible API).

## Features
- 🤖 AI advisor ("Priya") powered by Groq-hosted LLMs
- 🌐 10 Indian languages supported
- 🏗️ Multi-project management via sidebar
- 📋 Live lead capture (name, phone, budget, config)
- 💬 Contextual quick replies
- ➕ Add new projects via UI (no code required)

## Setup

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Set your Groq API key
```bash
# Linux / macOS
export GROQ_API_KEY=gsk_...

# Windows (Command Prompt)
set GROQ_API_KEY=gsk_...

# Windows (PowerShell)
$env:GROQ_API_KEY="gsk_..."
```

### 3. Run the app
```bash
python app.py
# or directly with uvicorn:
uvicorn app:app --reload --port 8000
```

Open your browser at **http://localhost:8000**

Interactive API docs (auto-generated by FastAPI) at **http://localhost:8000/docs**

## Adding Projects

Click **+ Add Project** in the sidebar. Fill in:
- Project name, developer, tagline, location
- Configurations (2 BHK | 950 sq ft | ₹2.5 Cr — one per line)
- Highlights (one per line)
- Amenities (comma-separated)
- Nearby landmarks (Name | Distance — one per line)

Projects are saved to `data/projects.json` and persist across restarts.

If `data/projects.json` is not present, the app automatically falls back to `projects.json` at the project root.

## API Endpoints
| Method | Route | Description |
|--------|-------|-------------|
| GET | `/` | Main chat interface |
| POST | `/api/chat` | Send message, get AI reply |
| GET | `/api/projects` | List all projects |
| POST | `/api/projects/add` | Add new project |
| DELETE | `/api/projects/{project_id}` | Remove a project |

## Lead Capture
The bot collects Name, Phone, Budget, and Config conversationally — it never forces the user. If a user declines or ignores a question, Priya moves on gracefully. Captured data appears in the sidebar panel in real time.
